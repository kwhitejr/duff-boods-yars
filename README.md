## Quick Start

#### Clone this repo

```bash
$ git clone https://github.com/epilande/yars.git
$ cd yars
```

#### Install dependencies

```bash
$ npm install
```

#### Launch front-end dev environment

```bash
$ npm run dev
```

#### Launch server

```bash
$ npm run server
```

#### Launch tests

```bash
$ npm run test
$ npm run test:watch
```
